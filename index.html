🔢 Binary — سیستەمی دوانی
STEP 01

Binary چییە؟

سیستەمی دوانی (Binary System) سیستەمێکی ژمارەیە کە تەنها دوو ژمارەی بەکاردێنێت: 0 و 1.

کومپیوتەرەکان هەموو داتایەک بە شێوازی ئەلکترۆنیکی خەزن دەکەن — ئەگەر کارەبا هەیە = 1 ، ئەگەر نییە = 0.

💡 هەر بیتێک (bit) کە تەنها دوو باری هەیە: 0 یان 1. 8 bit = 1 byte.
STEP 02

Bit و Byte

// جوری داتا 1 bit = 0 یان 1 4 bit = Nibble (نموونە: 1010) 8 bit = Byte (نموونە: 11001010) 16 bit = Word 32 bit = Double Word
STEP 03

خوێندنەوەی ژمارەی Binary

هەر پلەیەک بەرزبوونەوەی دوو چەندانە بە ئامادەی سفر:

// ژمارەی Binary: 1011 پلەی سێیەم → 1 × 2³ = 8 پلەی دووەم → 0 × 2² = 0 پلەی یەکەم → 1 × 2¹ = 2 پلەی سفر → 1 × 2⁰ = 1 کۆ: 8 + 0 + 2 + 1 = 11 (Decimal)
STEP 04

گواستنەوە: Decimal → Binary

ژمارەکە دابەش بکە لەسەر 2، ماوەکانی بنووسە:

// نموونە: 13 → Binary 13 ÷ 2 = 6 ماوە 1 6 ÷ 2 = 3 ماوە 0 3 ÷ 2 = 1 ماوە 1 1 ÷ 2 = 0 ماوە 1 // ماوەکان لە خوارەوە بۆ سەرەوە بخوێنەوە: 13₁₀ = 1101₂

// Identity Laws A + 0 = A A · 1 = A // Null Laws A + 1 = 1 A · 0 = 0 // Idempotent Laws A + A = A A · A = A // Complement Laws A + Ā = 1 A · Ā = 0 // Double Complement Ā̄ = A // De Morgan's Theorems A·B̄ = Ā + B̄ A+B̄ = Ā · B̄
STEP 04

سادەکردنەوەی ئەکسپرێشن

نموونە: F = A·B + A·B̄ بکەرە سادە:

F = A·B + A·B̄ = A·(B + B̄) ← Distributive Law = A·(1) ← Complement Law: B + B̄ = 1 = A ← Identity Law

STEP 05

De Morgan's Theorem

یاسای یەکەم: نافی (A AND B) = نافی A یان نافی B
‾(A·B) = Ā + B̄
یاسای دووەم: نافی (A OR B) = نافی A و نافی B
‾(A+B) = Ā · B̄
